СДЕЛАНО ПО ТЗ

1.  ТЗ п.1 «Первый экран»
    — Fullscreen hero (100vh) с фоновым Vimeo-видео (тот же ID 1061358748, что
      и у оригинала s7-kursk.ru).
    — Iframe растягивается под cover через width:177.78vh / height:56.25vw.
    — Видео приглушено: filter brightness 0.95 + saturate 0.85 + sepia 0.12,
      сверху тёплая полупрозрачная таупе-пелена → текст читается на любом кадре.
    — Эмблема «С7» (PNG 150×150, из ассетов клиники) и подпись
      «КЛИНИКА ЭСТЕТИЧЕСКОЙ МЕДИЦИНЫ» (Cormorant 36/400, uppercase).
    — CSS-спиннер крутится вокруг логотипа пока Vimeo не дал event 'play';
      после этого fade-out и удаление из DOM.
    — Preconnect на player.vimeo.com / i.vimeocdn.com → ускоренный старт видео.

2.  ТЗ п.2 «Единый шрифт» + блок «О клинике»
    — Cormorant (заголовки) + Manrope (тело) подключены через Google Fonts CDN.
    — Все H1-H3, параграфы, кнопки, подписи приведены к этим двум гарнитурам.
    — Блок «Добро пожаловать на сайт клиники С 7» с лидом про 14 000 клиентов
      и 4 преимуществами («Работают только эксперты», «Профессиональное
      оборудование», «Сертификаты», «Полный спектр услуг») — каждое с
      орнаментом «-II-» в Cormorant 16/2.5px letter-spacing.

3.  ДОП. — блок «О Центре» (не было в первом плане, добавлено отдельно)
    — Заголовок «О Центре» Cormorant 32/600.
    — Фото ресепшна клиники со «STUDIO SEVEN»-вывеской (CDN Tilda).
    — Лид + 4 абзаца основного текста про клинику.
    — Сворачиваемый текст: первый абзац виден всегда, остальные скрыты под
      кнопкой «Подробнее / Скрыть» (тот же data-collapsible-механизм).

4.  ТЗ п.3 «Услуги и цены» — ЛУКБУК-ВАРИАНТ
    Структура снята с реф-сайта s7.implatform.ru (через view-source).
    — 26 категорий, 202 услуги, реальные цены из прайса клиники.
    — ДЕСКТОП: 2-колоночный лукбук:
        • слева — вертикальный список 26 категорий (border-left:3px бронзовая
          полоска на активной),
        • справа — большая панель активной категории с описанием,
          SVG-росчерком в углу и таблицей услуг.
    — МОБИЛА: тот же layout сжимается в одну колонку, меню скроллится сверху.
    — ИЗЮМИНКИ:
        • Quick-chips фильтр сверху: Все / Инъекции / Аппаратная / Лазер /
          Эстетика — мгновенно скрывает неподходящие категории.
        • Прайс-калькулятор: чекбоксы на каждой услуге, сайдбар-карточка
          «Ваш выбор» считает count и total в реальном времени,
          показывает список выбранных услуг + «Ждём Вас в клинике!».
          Плавное появление/исчезновение через max-height + opacity transition.
        • SVG-росчерк в углу панели — декоративный бронзовый бренд-акцент.
        • Stagger-анимация: при смене категории строки услуг прилетают слева
          по очереди с задержкой 30 мс/строку.
    — Полный прайс-лист (PDF) → assets/price.pdf, табличный → assets/prices.pdf.

5.  ТЗ п.6 «Мы знаем, что беспокоит наших пациентов»
    — Стоит сразу после блока «Услуги» (как в оригинале).
    — 6 карточек, каждая: крупная цифра 1-6 фоном (Cormorant 180px,
      бежевый rgba(200,160,121,.18)), вопрос (Cormorant 22/500), ответ.
    — ДЕСКТОП: сетка 3×2 с большими отступами.
    — ПЛАНШЕТ: сетка 2×3.
    — МОБИЛА (≤768): горизонтальный слайдер по одной карточке + стрелки,

6.  ТЗ п.4 «Врачи» — горизонтальный слайдер
    — 8 карточек врачей: фото на всю карточку + оверлей с именем и спец.
    — Hover: фото zoom 1.05.
    — Десктоп 2 карточки в ряд, ультра-широкий 3, планшет 2, мобила 1.2
      (виден «хвост» следующей карточки — обязательное требование ТЗ).
    — Стрелки + индикаторы-точки (на десктопе) + свайп на тач.
    — sessionStorage запоминает позицию: при возврате со страницы врача
      слайдер открывается на той же карточке.
    — 8 отдельных страниц doctors/*.html с био и галереей сертификатов.

7.  ТЗ п.5 «Оборудование»
    — 4 карточки (Virtue RF, CAPELLO, CAPELLO Diode, ULTRAFORMER 3).
    — Per-card collapsible: лид-параграф всегда виден, остальной текст
      скрыт под кнопкой «Подробнее / Скрыть», нативный JS, плавная анимация
      max-height + opacity.
    — Кнопка в тёплых тонах (бронзовая обводка, hover — заливка бежевым).
    — Чередование text/image L-R, изображения 540×600 webp.

8.  ТЗ п.7 «Переход к фото всех врачей»
    — Блок «С7 / Команда» между Врачами и Оборудованием:

        • заголовок-разделитель «С7» (Cormorant 88/600),
        • fade-in анимация при появлении блока в viewport,
        • фото команды 2/3 ширины + текстовая колонка справа.
    — Декоративный SVG-«росчерк» позади текста, рисуется через
      stroke-dasharray draw-on когда блок входит в viewport (2.4s).
    — Использованы все 4 предложенных в ТЗ способа сглаживания «рывка».

9.  ТЗ п.8 «Бренды»
    — 11 логотипов партнёров (Juvederm, Ipsen, Filorga, Capello, Classys,

    — ДЕСКТОП: 6 колонок.
    — ПЛАНШЕТ: 4 колонки.
    — МОБИЛА (640): 3 колонки.
    — Узкая мобила (380): 2 колонки.

    — Логотипы по умолчанию ч/б с opacity 0.65, на hover — цветные scale(1.04).

ОБЩАЯ БАЗА

— Cormorant + Manrope (Google Fonts, weight 300-700, display=swap).
— CSS-переменные: --font-serif, --font-body, --container-max:1200px,
  --color-bg/text/footer-bg, --ease cubic-bezier(.4,0,.2,1).
— fade-in блоков при скролле через IntersectionObserver.
— Все слайдеры (доктора, услуги, беспокоит) обрабатывают:
  стрелки + точки (где есть) + touch-свайп + resize.
— `.no-js` fallback: если JS не загрузился, всё видно сразу.
— Sticky-header, sticky-cart (услуги-калькулятор), draw-on анимации.

КАК ОТКРЫТЬ

Сборщики не нужны. Двойной клик по index.html → страница открывается
в браузере. Шрифты, картинки и страницы врачей подтягиваются
относительными путями. Vimeo-видео и логотипы брендов берутся с CDN.
