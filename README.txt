ЧТО СДЕЛАНО
-----------

Полностью реализован блок «Услуги и цены» из ТЗ:

  Desktop (≥ 1024px)
    • Вертикальный список из 26 категорий слева (sticky, со скроллом
      если не помещается в viewport)
    • Справа — выбранная категория с заголовком и списком услуг
    • Активная категория подсвечена оранжевым (#ff8562) с левой
      полосой-маркером
    • Никакого горизонтального скролла, все категории видны
    • Никаких счётчиков «3 / 10» — убраны по ТЗ

  Mobile (< 1024px)
    • Категории-табы скрыты, виден только активный слайд
    • Слайдер по кнопкам ← / → (круглые, по краям)
    • В центре между стрелок — название текущей категории
      оранжевым, uppercase
    • Поддерживается свайп влево/вправо пальцем
    • Поддерживаются стрелки клавиатуры ← →
    • Один слайд = одна категория целиком (заголовок + все услуги)
    • Никаких точек или номеров слайдов — по ТЗ

  Анимация:
    • Fade-in при переключении категорий (translateY + opacity)
    • Плавные hover-эффекты на табах и услугах
    • Уважается prefers-reduced-motion

ЦИФРЫ
-----
  • Категорий: 26
  • Услуг: 202 (взяты с референса s7.implatform.ru)

СТЕК
----

  • HTML5 (semantic: <section>, <nav>, <article>, <main>, <header>)
  • CSS3 (Flexbox, Grid, custom properties, media queries)
  • Vanilla JS (без jQuery, без сборщика, без фреймворков)
  • БЭМ-нотация классов: блок__элемент--модификатор
  • Шрифты: Inter (текст) + Cormorant (заголовки) — Google Fonts

КАК ЗАПУСТИТЬ
-------------

Никаких сборщиков, npm, build-скриптов. Просто:

  1. Распаковать архив
  2. Открыть index.html в браузере (двойной клик)

Или через локальный сервер (рекомендуется для корректной работы
шрифтов):

  python3 -m http.server 8000
  # затем открыть http://localhost:8000

Тестировалось в Chromium 1440×900 (desktop), 900×1200 (планшет),
390×844 (мобильный).
