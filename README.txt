S7 Kursk — миграция сайта клиники эстетической медицины с Tilda
на нативный стек (HTML/CSS/JS). Хакатон.
=============================================================

Структура проекта (по ТЗ):

    ├── index.html
    ├── css/
    │   └── style.css
    ├── js/
    │   └── main.js
    └── assets/
        ├── fonts/
        └── images/

Что сделано
-----------

1. Блок «Оборудование» (id="equipment") — 1:1 с s7-kursk.ru:
   - Заголовок «Оборудование» — Cormorant 46 px, weight 600.
   - Подзаголовок Times New Roman 16 px.
   - 4 карточки с чередованием layout (текст слева/картинка справа,
     картинка слева/текст справа).
   - Заголовки карточек — Cormorant 26 px.
   - Первая карточка (Virtue RF) видна всегда.
   - Остальные 3 (CAPELLO, CAPELLO Diode, ULTRAFORMER 3) скрыты под
     катом, раскрываются кнопкой «Подробнее» — нативный JS, плавная
     CSS-анимация max-height + opacity.
   - Кнопка меняется на «Скрыть», иконка-шеврон поворачивается.
   - Доступно с клавиатуры (aria-expanded / aria-controls).

2. Footer — 1:1 с s7-kursk.ru:
   - Цвет фона #2c2a29, белый текст.
   - Уровень 1: «Контакты» (Cormorant 28 px) + телефоны + адрес +
     ссылки на парковки (синие, #0a00ff) + юр. блок (ООО, ИНН, ОГРН)
     + соцсеть VK + карта Яндекс справа.
   - Уровень 2: навигация (Главная / Политика / Структура / Вакансии
     / Цены) + крупный логотип «С7» (Cormorant 120 px) + строка
     лицензии + дисклеймер + предупреждение «Имеются противопоказания».
   - На десктопе колонки 5/7, на мобиле — стек в одну колонку.

3. Каркас:
   - Семантика: header / main / section / article / footer.
   - Подключён Cormorant с Google Fonts.
   - Чистый стек, без Tilda, без jQuery/React/Vue.
   - Адаптив: десктоп ≥ 1024 px, моб. ≤ 768 px.
   - Общие переменные в :root (палитра, шрифты, контейнер) — для
     согласованности с блоками других участников команды.

Что осталось доделать (для команды)
-----------------------------------

1. Дропнуть в assets/images/ фотографии оборудования с теми же
   именами файлов, что указаны в HTML:
       virtue-rf.png
       capello.jpg
       capello-diode.png
       ultraformer-3.jpg
   Можно сохранить с tildacdn.com или попросить у заказчика.

2. Если нужен полный текст оригинальных описаний оборудования
   и юр. дисклеймера — взять с https://s7-kursk.ru/ и вклеить
   в соответствующие <p> в index.html (структура уже готова).

3. Заменить плейсхолдеры в <main>:
   первый экран / о клинике / услуги / врачи / беспокойства / бренды.
   Каждый — отдельная задача участника команды.

4. Карта в футере — сейчас стоит ссылка yandex.ru/map-widget на
   приблизительные координаты ул. Радищева. Поправить точные.

Как открыть
-----------

Просто открыть index.html в браузере — сборщики не нужны (требование ТЗ).
